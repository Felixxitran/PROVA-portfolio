  • Each redesign improved how people moved through the site

Project 02 — Making a Slow Site Faster (and More Visible)

Codelynx: Grew Organic Traffic by 40%

The site was slow, wasn't showing up on Google, and people were dropping off. I fixed the speed issues, improved the SEO, and the traffic went up.

The Situation

The WordPress site was loading slowly, didn't look great on mobile, and the product pages weren't ranking for the keywords that mattered. They needed someone to dig into the technical side and also make the pages better at turning visitors into customers.

What I Did

  • Fixed the speed problems — images were too big and unoptimized, CSS was blocking the page from loading, and elements were jumping around as the page loaded (that's called layout shift)
  • Improved the SEO — rewrote meta titles and descriptions, added structured data so products look better in Google, fixed broken links, set up a proper sitemap
  • Worked on conversions — redesigned some landing pages, made the calls to action clearer, tested different layouts to see what worked
  • Set up Stripe for payments and made sure the site had SSL and a proper privacy policy
  • Set up tracking — Google Analytics and Search Console so we could actually see what was changing

SEO

How I'd think about SEO for a product like WithinStride

Here's what I'd actually focus on — not just buzzwords, but the stuff that moves the needle.

🎯

Finding the Right Keywords

I'd look for what people are actually searching — things like "smart insole for gait analysis" or "foot pressure sensor." Then match each keyword to the right page on the site.

📄

On-Page Basics

Each page gets its own meta title and description. Headings are organized properly. All images have alt text. Pages link to each other where it makes sense.

⚡

Making the Site Fast

Compress images, load things smartly so the page doesn't feel slow, make sure nothing jumps around while loading. All the stuff Google checks when ranking pages.

📊

Structured Data

This is the code that tells Google "this is a product" so it shows up nicely in search — with price, availability, reviews. Makes a big difference in click-through.

🔍

Tracking What Works

Set up Google Analytics and Search Console to see which pages are getting traffic, which keywords are ranking, and where people are dropping off.

🧪

Testing & Improving

Try different headlines, button placements, page layouts. See which version gets more clicks and go with the data instead of guessing.
